## 🚀 The "Slow-to-Fast" Guide: The SSD Miracle
If you bought an old laptop and it feels slow, **don't sell it.** 
In Pakistan, many "cheap" laptops are still sold with old HDDs (Hard Disk Drives). Replacing an HDD with a **SATA SSD** (Solid State Drive) is the single best upgrade you can make for under Rs. 5,000.
*   **HDD Boot Time:** 2 Minutes
*   **SSD Boot Time:** 15 Seconds
It literally makes a 2016 model laptop feel like a 2024 one.

## 🔍 Huzi's 2-Minute Testing Ritual
When you're at the shop, do this before paying:
1.  **Check the "White Spots":** Open a plain white image. If you see tiny bright dots, the screen is damaged. Ask for a discount or another unit.
2.  **The "Keyboard Test":** Open `keyboardtester.com` (using shop's Wi-Fi) and press EVERY single key. 
3.  **The Battery Report:** Ask the shopkeeper to show you the battery health using a software like "BatteryInfoView." If it's below 70%, ask for a new battery.

## 🙋 Frequently Asked Questions (FAQ)

### Is 8GB RAM enough for programming?
For basic Web Dev (HTML/CSS/JS) and Python, **Yes.** But if you want to run Android Studio or heavy Video Editing, you'll eventually need to upgrade to 16GB.

### Can I run PUBG on these laptops?
They can run it on **Lowest Settings**, but it won't be a great experience. These are "Work" machines, not "Gaming" machines.

### Should I buy a new "Local" Celeron laptop instead?
**Absolutely NO.** A used i5 business laptop from 2017 will be 3x faster and last 2x longer than a brand-new "Celeron" or "Pentium" laptop from a box.
